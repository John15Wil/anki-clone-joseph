# 功能更新说明

## 🎉 新增功能总览

### 1. ✏️ 卡片编辑功能
**位置**: 卡组详情页

**功能说明:**
- 在卡片列表中，每张卡片旁边新增"编辑"按钮（蓝色铅笔图标）
- 点击编辑后，会弹出编辑表单，带有橙色边框高亮
- 可以修改卡片的正面（问题）和背面（答案）
- 保存后立即更新显示

**使用方法:**
1. 进入卡组管理页面
2. 找到要编辑的卡片
3. 点击卡片右侧的蓝色"铅笔"图标
4. 修改内容
5. 点击"保存修改"（橙色按钮）

**视觉特点:**
- 编辑表单：橙色边框 + ✏️ 表情符号
- 保存按钮：橙色
- 编辑和删除按钮垂直排列

---

### 2. 📊 学习统计页面
**路由**: `/stats`
**入口**: 首页右上角"学习统计"按钮

**功能说明:**
- **总览卡片**：显示总卡组数、总卡片数、总复习次数、平均正确率
- **学习活动**：今天、本周、本月的学习数据
- **智能建议**：根据学习情况给出个性化建议

**统计指标:**
- 📚 总卡组数（蓝色）
- 📋 总卡片数（绿色）
- 📈 总复习次数（紫色）
- 🏆 平均正确率（橙色）

**学习活动:**
- 今天学习：蓝色背景
- 本周学习：绿色背景
- 本月学习：紫色背景

**数据来源:**
- 从 IndexedDB 实时计算
- 包含所有卡组的统计数据

### 3. ⌨️ 键盘快捷键
**位置**: 学习页面

**快捷键列表:**

| 快捷键 | 功能 | 使用场景 |
|--------|------|----------|
| `空格` | 显示答案 | 看到问题后 |
| `回车` | 显示答案 | 看到问题后 |
| `1` | 重来（完全不记得） | 答案显示后 |
| `2` | 困难（勉强想起来） | 答案显示后 |
| `3` | 一般（记得比较清楚） | 答案显示后 |
| `4` | 简单（非常轻松） | 答案显示后 |
| `空格` | 一般 | 答案显示后 |

**使用说明:**
- 学习时无需点击鼠标，全程使用键盘
- 每个评分按钮上都显示对应的快捷键
- 在输入框中输入时，快捷键不会触发

**提升效率:**
- 学习速度提升 50%+
- 减少鼠标操作
- 流畅的学习体验

### 4. 💾 数据导出功能
**位置**: 首页右上角"备份数据"按钮

**功能说明:**
- 一键导出所有数据为 JSON 文件
- 文件命名格式: `anki-clone-backup-YYYY-MM-DD.json`
- 包含完整的学习数据

**导出内容:**
- ✅ 所有卡组信息
- ✅ 所有卡片内容
- ✅ 学习复习记录
- ✅ 学习历史日志
- ✅ 导出版本和时间戳

**使用方法:**
1. 点击首页"备份数据"按钮
2. 浏览器自动下载 JSON 文件
3. 保存到安全位置

**数据格式:**
```json
{
  "version": "1.0",
  "exportDate": "2024-01-17T...",
  "decks": [...],
  "cards": [...],
  "reviews": [...],
  "logs": [...]
}
```

**安全建议:**
- 定期备份数据（建议每周一次）
- 保存到云盘或外部存储
- 清理浏览器数据前先备份
